# To-Do App

A simple to-do list application built with JavaScript.

## Features

- Add tasks to your to-do list.
- Mark tasks as completed.
- Delete tasks from your list.
- Filter tasks by status (All, Active, Completed).

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/tac3563/JS-Todo-App
   ```

2. Open `index.html` in your preferred web browser.

## Usage

- Enter a task in the input field and press Enter or click the "Add Task" button to add it to your to-do list.
- Click the checkbox next to a task to mark it as completed.
- Click the "X" button next to a task to delete it.
- Use the filter buttons to view tasks based on their status.

## File Structure

```
to-do-app/
  ├── index.html
  ├── styles.css
  └── app.js
```

- `index.html`: HTML structure of the application.
- `styles.css`: Styling rules for the app.
- `script.js`: JavaScript code for managing the to-do list functionality.
